# Primer día con git y github
```bash
git --version
git -v
git config --global user.name "micorreo@gmail.com"
git init , para crear un repositorio local
git status , para ver el estado de los archivos
git add . , para preparar todos los archivos a commit
git add nombreArchivo, para preparar un archivo en específico para commit
git commit -m "comentario relacionado al commit", hacer commit de los archivos preparados.
git log , para ver los commit existentes
```
